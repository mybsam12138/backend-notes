# Power BI Summary

## 1. What is Power BI?

Power BI is a Business Intelligence (BI) tool by Microsoft used to:
- Analyze data
- Build dashboards
- Create reports
- Share insights across teams

---

## 2. Components

### Power BI Desktop (Free)
- Local application
- Used to build reports and data models
- Supports:
  - Data modeling
  - DAX calculations
  - Data transformation (Power Query)

### Power BI Service (Web)
- Cloud-based platform
- Used for:
  - Sharing dashboards
  - Collaboration
  - Viewing reports

---

## 3. Typical Workflow

1. Build report → Power BI Desktop  
2. Publish → Power BI Service  
3. Share → Team accesses via web  

---

## 4. Key Features

- Drag-and-drop dashboard creation
- Data modeling (relationships, measures)
- Interactive filtering and drill-down
- Real-time data refresh
- Integration with multiple data sources

---

## 5. Languages Used

- DAX (Data Analysis Expressions) → calculations
- M (Power Query) → data transformation
- SQL → querying data sources

---

## 6. Pricing

### Free
- Power BI Desktop only
- No sharing

### Power BI Pro (~$10/user/month)
- Share dashboards
- Collaboration

### Power BI Premium
- Enterprise-level
- Better performance
- Viewers may not need license

---

## 7. Pros

- Powerful data analysis
- Easy for business users
- Rich visualization
- Strong Microsoft ecosystem integration

---

## 8. Cons

- Paid for collaboration
- Cloud dependency
- Limited modeling in web version
- Less suitable in restricted network environments

---

## 9. When to Use

Use Power BI when:
- You need business dashboards
- Non-developers need to explore data
- You want quick insights without coding

---

## 10. Summary

Power BI is a powerful BI tool that separates data analysis from application development.  
Developers prepare data, while analysts build dashboards independently.

---

## 11. Interview Answer

Power BI is a business intelligence platform that allows users to connect to data sources, model data, and build dashboards. It typically uses a desktop application for development and a web service for sharing and collaboration.
