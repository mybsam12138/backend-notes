# Vector – Definition Summary

## What is a Vector?

**A vector is an ordered list of numbers that represents a point or direction in space.**

---

## Key Ideas

- A vector is a **coordinate in space**
- It can represent:
  - a **position**
  - a **direction**
- It has:
  - **magnitude (length)**
  - **direction**

---

## Examples

### 2D Vector
```
[3, 4]
```
Represents a point or direction in 2D space.

---

### 3D Vector
```
[1, 2, 3]
```

---

### High-Dimensional Vector (Embedding)
```
[0.12, -0.45, 0.88, ...]
```
Used to represent meaning in AI systems.

---

## One-Line Takeaway

**A vector is a mathematical representation of data as a point in space.**
